# テスト用アカウント

開発・テスト用のアカウントです。これらのアカウントはパスワードなし（または簡単なパスワード）でログインできます。

## テストアカウント一覧

| 役割 | メールアドレス | パスワード | ダッシュボード |
|------|---------------|-----------|---------------|
| モニター | `monitor@test.com` | `test1234` | `/dashboard` |
| クライアント | `client@test.com` | `test1234` | `/client` |
| 管理者 | `admin@test.com` | `test1234` | `/admin` |
| サポート | `support@test.com` | `test1234` | `/support` |

## 使用方法

1. ログインページ（`/login`）にアクセス
2. 上記のメールアドレスとパスワードを入力
3. パスワードが空でもログインできます（テスト用アカウントの場合）
4. 各役割に応じたダッシュボードにリダイレクトされます

## 注意事項

- これらのアカウントは開発・テスト環境専用です
- 本番環境では使用しないでください
- パスワードは簡単なため、セキュリティには注意してください

## アカウントの作成

初回ログイン時に、これらのアカウントが自動的に作成されます。もし作成されない場合は、Supabase Dashboardで手動で作成してください。

作成手順：
1. Supabase Dashboardにアクセス
2. 「Authentication」→「Users」を開く
3. 「Add user」→「Create new user」を選択
4. メールアドレスとパスワードを入力
5. 「User Metadata」に `role` を追加（例: `{"role": "monitor"}`）
6. 「Auto Confirm User」を有効化
